## Installation

To set up the project locally, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/quready.git
   ```

2. Navigate to the project directory:

   ```bash
   cd quready
   ```

3. Install dependencies:

